# Dreamorphine
http://stone.am0200.com

A growing collection of album covers.

## How it works

- Install node modules

```bash
# install modules
$ npm i
```

- Upload album info

```bash
$ node server.js
```

access `http://127.0.0.1:2333` to add new album

- Publish the albums

```bash
# build
$ gulp
# push to Github
$ git add/push
```

access `http://127.0.0.1:2222` for local test
